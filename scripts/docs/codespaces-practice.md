## Understanding Git Workflow
Explain in 2–3 sentences the difference between:
- Local workspace (Codespace)
- Remote repository (GitHub.com)
A local workspace, like GitHub Codespaces, is where you work on the project directly—writing, editing, and testing the code in your own development environment. The remote repository on GitHub is the online version of the project that stores the official code and its history, allowing you to push your changes and collaborate with others.
